# PlasticFEM — 2D 塑性加工シミュレーションソフトウェア 製品概要

**バージョン**: 0.3.0 (v3)
**更新日**: 2026年6月13日
**単位系**: mm / N / MPa

---

## 1. 製品コンセプト

PlasticFEM は、鍛造・押出し・据え込み・曲げ等の **金属塑性加工プロセスを対象とした 2D 有限要素解析ソフトウェア**です。軸対称・平面ひずみの弾塑性大変形解析を行い、DEFORM-2D 相当の工学的精度を、**Python ベースの完全オープンアーキテクチャ**で提供します。

DEFORM・MARC 等の商用コードをお使いの方向けに、本書は採用されている**定式化・数値手法・検証結果・現状の制約**を明示します。

## 2. 定式化（CAE 技術者向け仕様）

### 2.1 連続体・運動学

| 項目 | 採用方式 |
|------|---------|
| 解析次元 | 2D：軸対称（r-z）/ 平面ひずみ |
| 大変形 | Updated Lagrange（増分ごとに参照配置を更新する hypoelastic 定式化）|
| 構成則 | J2（von Mises）弾塑性 + 等方硬化 |
| 応力更新 | 陰的 Radial Return Mapping（Simo & Hughes 1998, Box 3.2）|
| 整合接線 | 一貫アルゴリズム接線剛性 D^alg（有限差分検証で相対誤差 ~1×10⁻⁶ を確認）|
| ひずみ表現 | 4成分 Voigt [εrr, εzz, εθθ, γrz]（軸対称の周方向ひずみ εθθ = u_r/r を含む）|

### 2.2 非線形解法

| 項目 | 採用方式 |
|------|---------|
| 平衡反復 | Newton-Raphson（整合接線）+ 残差ノルム基準（力スケール相対 1×10⁻⁴）|
| 接触解法 | アクティブセット法（外ループで接触集合を固定 → 内 Newton 収束 → 集合更新）|
| 増分制御 | 適応サブステップ（収束失敗時は増分を1/2、回復後に漸増）|
| 安定化 | 振動検出による減衰 Newton、増分ごとの補正量キャップ |
| 線形ソルバ | スパース直接法（SciPy / SuperLU）|
| 高速化 | 全要素一括の塑性計算・剛性組立（NumPy ベクトル化、einsum）、増分内 B 行列キャッシュ |

> **計算速度の目安**：後方押出し（軸対称・約700節点・20mmストローク・約400ステップ）で、解析専用PC で約3分、8年前の Core i5 で約7分（いずれもシングルスレッド）。

### 2.3 要素技術

| 項目 | 内容 |
|------|------|
| 要素 | 3節点三角形（CST, 1次・定ひずみ）|
| 数値積分 | 1点（重心）|
| 体積ロッキング対策 | **F-bar（節点平均体積ひずみ法）を実装中**（§7 参照）|

### 2.4 接触・摩擦

| 項目 | 内容 |
|------|------|
| 工具 | 剛体（DXF 閉多角形）。複数工具同時。各セグメントの外向き法線を符号付き面積から自動決定 |
| 接触判定 | 節点-セグメント、閉多角形の内外判定による貫入検出（DEFORM 流の貫入トレランス許容）|
| 法線方向 | ペナルティ法（支配面積重み付け：fn = kn·A·δ、メッシュ非依存の接触圧）|
| 貫入制御 | 貫入深さがトレランス超過でリメッシュ＋表面投影（素通り防止）|
| 摩擦モデル | **クーロン摩擦 τ = μ·p** と **せん断摩擦 τ = m·k**（DEFORM 互換）の両対応、複合則 τ = min(μp, mk) も可 |
| 摩擦正則化 | tanh によるスティック-スリップ連続化 |
| 工具運動 | 変位制御（パンチ・ストローク）/ **力制御**（§2.5）|

### 2.5 力制御工具（板押さえ・逆押さえ）

| モード | 指定 | 用途 |
|--------|------|------|
| 固定力 | f_const [N]（例：1000N固定）| 板押さえ・逆押さえ（カウンターパンチ）|
| ばね圧 | k_spring [N/mm]（例：200N/mm）| ばね式板押さえ |
| 変位制御 | vel（ストローク比）| ラム同期の逆押さえ等 |

**モノリシック連成**：力制御工具の位置を未知数として系（[[A,−C],[−Cᵀ,dss]]）に組み込み、
材料変形と工具位置を同時求解。これにより、工具移動と逆向きにバネが撓む back-pressure
（逆押さえ）でも、完全拘束された非圧縮材料下で安定に解ける（交互更新では発散する難条件）。
閉塞鍛造で力制御逆押さえが正常完走することを確認済み。

### 2.6 メッシュ・リメッシュ

| 項目 | 内容 |
|------|------|
| メッシュ生成 | Gmsh（Frontal-Delaunay）|
| 適応リメッシュ | アスペクト比・要素反転・貫入深さで自動トリガ |
| 局所細分化 | パンチ作用部・高ひずみ域（PEEQ 分布上位）を選択細分化 |
| 状態マッピング | 旧要素重心からの線形補間（凸包外は最近傍フォールバック）|
| リメッシュ後処理 | ゼロストローク緩和ステップ、境界の工具面スナップ |

### 2.7 スプリングバック

工具を全除去し、保持していた反力 R0 を λ:1→0 で**徐々に解放**する増分リリース方式（減衰 Newton）で自由弾性回復を計算。塑性状態は凍結し逆降伏も追跡。単一工程・多段いずれの末尾でも適用可（`--springback`）。

### 2.8 材料モデル

| モデル | 式 |
|--------|----|
| 区分線形 | 真ひずみ–流動応力テーブル（任意点数、線形補間）|
| 冪乗則（n値硬化）| σ = C·(ε0 + εp)ⁿ |
| Voce | σ = σ∞ − (σ∞ − σ0)·exp(−k·εp) |

弾性は等方（ヤング率 E・ポアソン比 ν）。

## 3. 入力 / 出力

### 入力
- **工具・材料形状**：DXF（LINE / ARC / LWPOLYLINE / POLYLINE / CIRCLE）。ARC・バルジは折れ線離散化し端点一致で輪郭をチェイン化、最大閉ループを材料輪郭、閉多角形を工具とする
- **材料物性**：E, ν, 硬化則（n値・強度定数・流動応力テーブル）
- **境界条件**：パンチ移動方向・ストローク、各工具の摩擦係数（μ または m）、板押さえ／逆押さえ力（固定／ばね）

### 出力
- von Mises 相当応力・相当塑性ひずみ（PEEQ）コンター（PNG、**全工程固定カラースケール**で工程間比較可）
- **フローネット（ファイバーフロー）**: 材料追従トレーサ格子の変形で材料流れ・巻き込み（ラップ）・内部欠陥を可視化（DEFORM flownet 相当）
- 荷重-ストローク曲線（CSV / PNG、DEFORM 参照値オーバーレイ可）
- 全ステップデータ（HDF5）：座標・要素・応力・PEEQ・Mises
- step0（初期状態）および各工程先頭フレームを保存

## 5. 商用ソフトとの精度比較（後方押出し・S45C・20mm）

DEFORM-2D（せん断摩擦 m=0.12、流動応力テーブル同一）との比較：

| ストローク | DEFORM | PlasticFEM | 差異 |
|----------|--------|-----------|------|
| 0.5 mm | 1,115 kN | 1,196 kN | +7.3% |
| 5.0 mm | 1,392 kN | 1,417 kN | **+1.8%** |
| 10.0 mm | 1,470 kN | 1,428 kN | **−2.9%** |
| 20.0 mm | 1,483 kN | 1,556 kN | +4.9% |

- 荷重-ストローク：実効 **±5%以内**（リメッシュ時の鋸歯状変動を除く）
- 変形形状・材料流れ・応力分布パターン：DEFORM と一致
- **摩擦モデルは要一致**：DEFORM のせん断摩擦 m に対しクーロン μ を用いると高接触圧域で荷重を過大評価するため、比較時は同一の摩擦則（m）を使用

## 7. 現状の制約と開発ロードマップ

CAE 技術者向けに、既知の制約と対応方針を明示します。

| 項目 | 現状 | 対応方針 |
|------|------|---------|
| 薄板成形（V曲げ）| **解決済み** — 相対劣化リメッシュトリガで薄板要素を自動許容し完走 | ― |
| 薄板成形（曲面パンチ＋ばね板押さえ）| 曲面剛体工具との接触チャタリングで低速・不安定（最優先の残課題）| 接触面平滑化（曲面の厳密ギャップ計算）、板厚方向メッシュ |
| 力制御工具のback-pressure | 完全拘束系での発散は**モノリシック連成で解決済み**（閉塞鍛造の力制御逆押さえ完走）| ― |
| 体積ロッキング（CST）| 当初 closed_die 停滞の主因と疑ったが、真因は力制御連成だった（F-bar で改善せず）。自由表面系では非顕在 | F-bar は実装済（整合接線つき）だが自由表面系で過軟化し既定OFF。要再検証 |
| **リメッシュ時の荷重変動** | 状態マッピング（線形補間）に起因する鋸歯状変動（移動平均で除去可）| 高次保存的マッピング |
| **要素次数** | 1次三角形のみ | 2次要素 / F-bar による精度向上 |
| 熱連成（温間・熱間）| 断熱昇温モデルのみ | 熱-力連成 |
| 破断系（せん断・ファインブランキング）| 未対応 | ダメージソフトニング（Cockcroft-Latham 等）＋軟化の収束正則化 |
| 3D 解析 | 未対応（2D 専用）| 中長期 |
| GUI / STEP 入力 | 開発中 | — |

> **設計思想**：検証済みの機能（押出し・据え込み・鍛造・多段・固定力工具・スプリングバック）は安定動作を最優先で維持し、新機能（F-bar 等）はフラグで分離して既存検証を壊さない方針を採っています。

## 8. 特長

1. **完全オープンアーキテクチャ** — ソースを保有し、構成則・接触・摩擦・出力を自由に拡張。社内材料DB・社内ワークフローと統合可能。
2. **DEFORM 互換の摩擦モデル** — せん断摩擦 m と クーロン μ の両対応で、既存 DEFORM モデルとの直接比較が可能。
3. **DXF 直接入力** — CAD 設計工具をそのまま読み込み。
4. **多段シナリオ＋スプリングバック** — 工程間の状態引き継ぎと自由弾性回復。
5. **サーバ運用を見据えた堅牢性** — 非収束時に部分結果を返す打ち切りガード等、無人ジョブ実行に必要な安全機構を実装。
